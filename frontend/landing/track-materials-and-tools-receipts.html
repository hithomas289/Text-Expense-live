Common Questions

How do I separate materials from tools for taxes?

Add "materials" or "tools" notes when sending receipts. Filter by category when exporting for tax prep.

Can I track tool serial numbers?

Yes. Add serial numbers as notes when sending tool purchase receipts for theft or warranty tracking.

What if I return materials?

Send the return receipt too. Note "return - [original purchase date]" to keep records accurate.

Does it track tool depreciation?

Send purchase receipts with costs. Your accountant can use the data for depreciation schedules.

Can I find receipts for warranty claims years later?

Absolutely. All receipts stored permanently. Search by date range, amount, or supplier to find any purchase.
